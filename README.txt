STORIA 52 - WEBAPP

Apri index.html oppure pubblica i file su GitHub Pages.

Funzioni principali:
- tema carte chiaro o scuro, con preferenza salvata;
- generazione casuale oppure selezione manuale di protagonista, situazione, problema e obiettivi;
- modalità singolo telefono con elenco giocatori persistente;
- modalità multi-telefono tramite inviti progressivi numerati automaticamente;
- archivio completo delle carte;
- utilizzo offline dopo il primo caricamento.

Multi-telefono senza backend:
- il telefono host crea la stanza;
- genera in ordine l’invito del giocatore 1, poi 2, poi 3;
- numero, incipit e obiettivo sono già inclusi nel link privato;
- chi riceve il link non deve inserire il proprio numero.
