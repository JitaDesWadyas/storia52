STORIA 52 - WEBAPP

Apri index.html oppure pubblica i file su GitHub Pages.

Modalità multi-telefono:
- tutti usano lo stesso codice partita;
- ogni giocatore sceglie un numero diverso;
- lo stesso codice genera lo stesso incipit pubblico;
- il numero giocatore genera un Obiettivo segreto diverso.

La webapp è installabile e funziona offline dopo il primo caricamento.
